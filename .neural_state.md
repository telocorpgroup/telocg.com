## ⚠️ NOTAS IMPORTANTES

1. **Para activar Auth admin:** Ejecutar `supabase-schema-migration-v4.sql` → crear usuario admin en Supabase Dashboard → Authentication → Users → Add user (email/password).
2. **Ruta del workspace tiene caracteres especiales** — para deploy Edge Functions, copiar a `C:\temp\sb-deploy\supabase\`.
3. **El .env local tiene BOM encoding** — renombrar a `.env.bak` antes de `supabase functions deploy`.
4. **Imágenes de productos** en `TeloCorp/images/image1-71.png` servidas via CDN wsrv.nl → WebP.
5. **Google Maps ahora lazy-load** — solo se carga cuando se navega a TeloLleva.
