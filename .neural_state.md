## ⚠️ NOTAS IMPORTANTES

1. **Ruta del workspace tiene caracteres especiales** (á, é) que causan problemas con Docker. Para deploy de Edge Functions, copiar `supabase/` a `C:\temp\sb-deploy\supabase\` y ejecutar desde ahí.

2. **El .env local tiene BOM encoding** que rompe la CLI de Supabase. Renombrar a `.env.bak` antes de `supabase functions deploy`, luego restaurar.

3. **Imágenes de productos** están en `TeloCorp/images/image1.png` a `image22.png` (cada una ~1.2MB PNG). Se sirven via CDN wsrv.nl que convierte a WebP automáticamente (~50-80KB final).

4. **Sincronización tienda ↔ Supabase:** `script.js` intenta leer productos de Supabase al cargar. Si Supabase no responde, usa el catálogo hardcodeado como fallback.

5. **CardNet/Azul (pagos con tarjeta en RD):** Requieren afiliación comercial presencial. La arquitectura soporta integración futura via Edge Function.

6. **paypal.me no carga** — el flujo actual envía al usuario a WhatsApp con instrucciones para pagar a telocorpgroup@gmail.com
