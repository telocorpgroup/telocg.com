# TELOCG

Repositorio del sitio web de `telocg.com`.

## Despliegue con GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Conecta este repositorio local con el remoto.
3. Haz push a la rama principal.
4. En GitHub, activa Pages desde `Settings > Pages`.
5. Usa `telocg.com` como dominio personalizado.

El archivo `CNAME` ya esta configurado para GitHub Pages.
